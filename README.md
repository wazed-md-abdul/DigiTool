## 🚀 About The Project

DigiTool is a modern, responsive digital tool marketplace built with React and Vite. It provides users with a seamless experience to explore, purchase, and manage digital tools with features like cart functionality, pricing plans, and interactive UI components. The application emphasizes clean design, performance, and user experience.

### 🌟 Key Features

- **🛒 Cart Management**: Add items to cart, view cart summary, and proceed to checkout
- **💳 Pricing Plans**: Multiple subscription tiers with detailed features
- **🔍 Product Exploration**: Browse through various digital tools with detailed descriptions
- **📱 Responsive Design**: Optimized for desktop and mobile devices
- **🔔 Toast Notifications**: Real-time feedback for user actions
- **🎨 Modern UI**: Built with Tailwind CSS and DaisyUI for a sleek interface

## 🚀 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/wazed-md-abdul/DigiTool.git
   cd DigiTool
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview the production build**
   ```bash
   npm run preview
   ```
